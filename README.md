# OTT Content Popularity Prediction

## Problem Statement
Predict whether OTT content will have High, Medium 
or Low popularity based on content metadata.

## Dataset
2500 movies and series from Netflix and Prime Video

## Models Used
Decision Tree, KNN, Random Forest, SVM, Logistic Regression

## Best Model
Random Forest — 39% Accuracy

## Key Finding
Content metadata alone is insufficient for popularity 
prediction. Data leakage was identified and handled.

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Google Colab
